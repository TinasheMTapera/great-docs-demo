I’m creating a function below:

``` python
def myfunc(arg1, arg2, **kwargs):
    """
    Example function.

    Parameters
    ----------
    arg1 : int
    arg2 : str

    Returns
    -------
    None
    """

```

`arg2`, and returns their sum. The function is decorated with a
docstring that describes its parameters and return value.
